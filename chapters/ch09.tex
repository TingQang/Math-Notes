\chapter{向量与线性方程组}
\label{ch:lin_vec_eq}
\begin{chaptergoal}
    本章主要介绍向量的基本概念及其运算，线性方程组的表示与求解方法。通过学习本章内容，读者应掌握以下知识点：
    \begin{itemize}
        \item 理解$n$维向量的概念，掌握向量的线性运算（加法、数乘）、理解向量的线性组合与线性表示的概念
        \item 理解向量组线性相关、线性无关的概念，掌握向量组线性相关、线性无关的判定方法（定义法、秩判别法、行列式判别法）
        \item 理解向量组的极大线性无关组和向量组的秩的概念，掌握极大线性无关组的求解方法和向量组秩的计算
        \item 了解向量组等价的概念，了解矩阵的秩与其行（列）向量组的秩的关系
        \item 理解齐次线性方程组有非零解的充分必要条件，理解非齐次线性方程组的通解的求解方法（初等行变换法）
        \item 了解$n$维向量空间、基、维数、坐标的概念，了解基变换与坐标变换公式
    \end{itemize}
\end{chaptergoal}
