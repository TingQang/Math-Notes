\chapter{行列式与矩阵}
\label{ch:lin_alg_det_mat} 
\begin{chaptergoal}
    本章我们将学习行列式与矩阵的基本概念及其运算方法，掌握行列式的性质和计算技巧，理解矩阵的基本运算和类型，为后续章节的线性方程组和特征值问题打下基础。本章重点如下：
    \begin{itemize}
        \item 了解行列式的概念，掌握行列式的性质，熟练掌握$n$阶行列式的计算方法（展开方法、化上三角/下三角、递推法、范德蒙德行列式）
        \item 理解矩阵的概念，掌握矩阵的线性运算（加法、数乘）、乘法运算、转置运算，了解矩阵运算的性质
        \item 理解逆矩阵的概念，掌握逆矩阵的性质，熟练掌握逆矩阵的求解方法（伴随矩阵法、初等行变换法），能利用逆矩阵求解矩阵方程
        \item 理解矩阵的初等变换概念，掌握矩阵的性质，了解矩阵的秩与初等变换的关系
        \item 理解矩阵的秩的概念，掌握矩阵秩的计算方法，了解矩阵秩的性质
        \item 了解分块矩阵的概念，掌握分块矩阵的运算法则，能利用分块矩阵简化计算
    \end{itemize}
\end{chaptergoal}
