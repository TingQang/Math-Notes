\chapter{随机变量及其分布}
\label{ch:prob_dist}
\begin{chaptergoal}
    \begin{itemize}
        \item 理解随机变量的概念，了解离散型随机变量、连续型随机变量的概念
        \item 掌握离散型随机变量的分布律（概率质量函数）的概念及性质，熟练掌握常见离散型分布：0-1分布、二项分布、泊松分布、几何分布、超几何分布
        \item 掌握连续型随机变量的概率密度函数的概念及性质，熟练掌握常见连续型分布：均匀分布、正态分布、指数分布，能利用概率密度计算概率
        \item 理解随机变量的分布函数的概念和性质，能利用分布律/概率密度求分布函数，反之亦然
        \item 掌握随机变量函数的分布的求解方法（离散型：直接法 ；连续型：分布函数、公式法
        \item 理解二维随机变量的概念，掌握二维离散型随机变量的联合分布律、边缘分布律、条件分布律；掌握二位连续型随机变量的联合概率密度、边缘概率密度、条件概率密度
        \item 理解随机变量独立性的概念，掌握二维随机变量独立的判定方法
        \item 理解随机变量的数学期望、方差、标准差的概念，掌握其性质及计算方法；掌握常见分布的数学期望和方差
        \item 理解协方差、相关系数的概念，掌握其性质及计算方法，了解相关性与独立性的关系
        \item 了解切比雪夫不等式、大数定律、中心极限定理的内容，能利用中心极限定理近似计算概率
    \end{itemize}
\end{chaptergoal}
