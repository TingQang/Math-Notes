\chapter{无穷级数}
\label{ch:calc_series} 
\begin{chaptergoal}
    本章目标是掌握无穷级数的基本概念、收敛性判别方法及其应用，重点包括：
    \begin{itemize}
        \item 理解常数项级数收敛、发散的概念，掌握级数收敛的必要条件和基本性质（线性性、加括号性质
        \item 掌握正项级数收敛性的判定方法（比较判别法、比值判别法、根值判别法、积分判别法）
        \item 掌握交错级数的莱布尼兹判别法，理解绝对收敛与条件收敛的概念及关系
        \item 理解幂级数的概念，掌握幂级数的收敛半径、收敛区间、收敛域的求解方法
        \item 掌握幂级数的四则运算法则、逐项求导、逐项积分性质，能将函数展开为幂级数（泰勒级数），能利用幂级数求常数项级数的和
        \item 了解傅里叶级数的概念，掌握周期为$2\pi$的傅里叶级数展开式，了解狄利克雷收敛定理；掌握周期为$2l$的傅里叶级数展开式
    \end{itemize}
\end{chaptergoal}
